# Icons Assets

Đặt các file icon vào đây.

## Cấu trúc đề xuất:

```
icons/
├── ic_customer.png           # Icon cho Customer App
├── ic_staff.png             # Icon cho Staff App
└── ic_admin.png             # Icon cho Admin App
```

## Lưu ý:

- Sử dụng định dạng PNG với background transparent
- Chuẩn bị nhiều kích thước (1x, 2x, 3x) nếu cần
- Đặt tên file theo snake_case với prefix `ic_`
