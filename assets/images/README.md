# Images Assets

Đặt các file ảnh vào đây.

## Cấu trúc đề xuất:

```
images/
├── logo.png                  # Logo ứng dụng
├── placeholder.png           # Ảnh placeholder cho món ăn
├── empty_cart.png           # Icon giỏ hàng trống
└── empty_orders.png         # Icon không có đơn hàng
```

## Lưu ý:

- Sử dụng định dạng PNG hoặc JPG
- Tối ưu kích thước ảnh trước khi thêm vào project
- Đặt tên file theo snake_case (ví dụ: `logo_customer.png`)
